Assignments

Assignment 1 - Basic HTML

In this assignment, we were tasked to create a website about any topic, to get comfortable with basic HTML functions. I chose one of my favorite recipes as the topic.

Assignment 2 - Basic CSS

In this assignment, we were tasked to create a website about any topic and style it with basic CSS elements. I chose loteria as the topic and included an example game.

Assignment 3 - Page Layout

In this assingment, we were tasked to follow a given layout to get comfortable with using flexbox.

Assignment 5 - Recreate CSS Page

In this assingment, we were tasked to recreate an existing webpage in order to hone our flexbox skills.

Assignment 6 - JavaScript, Buttons, Functions, and more

In this assingment, we were tasked to create a page that contained a hide and seek picture, a moving image, and a comment section to practice with JavaScript.

Assignment 7 - Conditionals

In this assingment, we were tasked to create a page that contained a comparison of ages and a thermometer to measure a fundraiser goal in order to practice our JavaScript skills further.
